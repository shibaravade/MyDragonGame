Welcome to DragonGame

Below are the steps for running the application.

1) install node.js on your system
   after installation--> open node command prompt

2) install typescript by giving command
   npm install typescript -g

3) install angular/cli by giving command
   npm install @angular/cli -g

4) create project using angular/cli
  ng new <project-name>

5) replace files and folders app, assets, index.html, angular.json, package.json 
   delete package-lock.json file which already created.

6) go inside project folder on node command prompt where package.json is present,

Example: D:\DragonGame>

give command
npm install

.......after installation..... 

give command
ng serve -o
